#  Urban Mobility & Economic Productivity Analysis (LATAM 2024)

###  English Description
**Project Overview:**
This project investigates the correlation between **traffic congestion** (delays, jam lengths) and **economic indicators** (GDP per capita) across 15 major Latin American cities. The goal is to determine if economic productivity is a direct driver of urban traffic congestion.

**Key Features:**
* **Data Cleaning:** Type conversion, string normalization, and handling of missing values.
* **Data Aggregation:** Calculation of annual traffic averages per city using `groupby()`.
* **Data Integration:** Merging disparate datasets (Traffic & Economy) via Inner Joins.
* **EDA:** Visualization of relationships using Matplotlib and Seaborn.

**Key Finding:**
The analysis revealed **no direct linear correlation** between high GDP and traffic delays. For instance, **Montevideo** (highest GDP in sample) shows minimal delays compared to **Mexico City**, suggesting that infrastructure plays a larger role than wealth alone.

**Tech Stack:** `Python` `Pandas` `Matplotlib` `Seaborn`
